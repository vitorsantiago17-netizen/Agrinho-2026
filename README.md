# 🌱 EcoAgro Smart Farm | Agro Forte, Futuro Sustentável

Uma plataforma web interativa que demonstra como a tecnologia pode contribuir para uma agricultura mais eficiente, sustentável e responsável. O projeto apresenta indicadores ambientais, simulação de irrigação inteligente e recursos modernos de interface para conscientizar sobre a importância do equilíbrio entre produção agrícola e preservação ambiental.

---

## 🎯 Objetivo do Projeto

Demonstrar como soluções tecnológicas podem auxiliar o agronegócio na utilização consciente dos recursos naturais. O EcoAgro Smart Farm busca mostrar a importância da inovação para reduzir desperdícios, aumentar a eficiência produtiva e promover práticas sustentáveis no campo.

---

## 🖥️ Recursos Técnicos

### 📊 Dashboard Inteligente

O sistema apresenta indicadores simulados relacionados à sustentabilidade agrícola:

* 💧 Nível de Água
* ☀️ Energia Solar
* 🌱 Qualidade do Solo
* 🌍 Emissão de Carbono

Os indicadores são animados dinamicamente utilizando JavaScript, proporcionando uma experiência mais interativa ao usuário.

### 🚜 Irrigação Inteligente

Ao ativar o sistema de economia:

* Atualização automática dos indicadores.
* Simulação da redução do desperdício de água.
* Demonstração do impacto positivo da tecnologia no agronegócio.

### 📩 Formulário de Contato

O site possui um formulário funcional com:

* Validação de campos obrigatórios.
* Limpeza automática após envio.
* Mensagem de confirmação para o usuário.

### 🌙 Modo Escuro (Dark Mode)

Recurso desenvolvido em JavaScript que permite:

* Alternância entre tema claro e escuro.
* Melhor conforto visual.
* Interface moderna e acessível.
